## 製作筆記

1. 先想好階層，定義資料欄位。

2. 依據資料欄位，設置 Ref 製 Vue。

3. 測試更改欄位資料，畫面是否會如預期的變動。

4. 新增方法
- 使用`二維陣列`算出父組件和子組建的關係
- 父層只要知道小孩是
- 子層則只需要管好自己的值
nodes.
